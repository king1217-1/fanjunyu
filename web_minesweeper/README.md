# 扫雷网页小游戏与 C++ 控制台版本

本项目包含两个部分：

1. 浏览器中运行的 **网页扫雷小游戏**（HTML + CSS + JavaScript）
2. 使用 C++ 编写的 **控制台扫雷游戏逻辑**，支持相同的难度与自定义规格

## 目录结构

- `index.html`：网页入口
- `style.css`：页面样式
- `script.js`：网页扫雷逻辑（浏览器交互）
- `minesweeper.cpp`：C++ 控制台版扫雷实现
- `victory.mp3`（可选）：胜利音效文件（需自行放入同目录）

## 一、运行网页扫雷小游戏

1. 使用浏览器打开 `index.html` 即可开始游戏。
2. 玩法说明：
   - 在顶部选择难度：
     - 简单：5 × 5
     - 普通：9 × 9
     - 困难：17 × 17
     - 自定义：支持最大 500 × 500
   - 点击“开始游戏”重新生成棋盘。
   - **左键**：打开格子。
   - **右键**：标记或取消标记“旗”（认为这里有雷）。
   - 顶部会显示计时器以及“剩余雷数”。
   - 清除所有非雷格子后会停止计时并显示胜利提示，同时尝试播放 `victory.mp3`。

### 胜利音效

如果希望在胜利时播放音效：

1. 自行准备一段短的 mp3 音效文件，例如 `victory.mp3`。
2. 将文件放在与 `index.html` 同一目录下。

## 二、运行 C++ 控制台扫雷

1. 在终端进入本目录，例如：

   ```bash
   cd /d d:\小游戏\web_minesweeper
   ```

2. 使用任意支持 C++17 的编译器编译：

   ```bash
   g++ minesweeper.cpp -std=c++17 -O2 -o minesweeper
   ```

3. 运行：

   ```bash
   ./minesweeper
   ```

4. 控制台玩法说明：
   - 先选择难度（1~4，自定义最大 500 × 500）。
   - 每一步输入：`行 列 操作`
     - `操作` 为 `o`（打开）或 `f`（标记/取消标记）。
     - 例如：`3 4 o` 表示打开第 3 行第 4 列；`2 1 f` 表示标记第 2 行第 1 列为雷。
   - 游戏会记录从第一次落子开始到结束的总用时，在胜利时显示。

> 提示：网页版负责“页面 + 鼠标交互 + 计时”，核心扫雷算法在两种实现（JS 和 C++）中是一致的，你可以根据需要重点学习 `minesweeper.cpp` 中的 C++ 实现逻辑。
