请在本目录中放入两个图标文件：

1. 旗帜图标：命名为 flag.svg
   - 可以从以下任意地址下载自己喜欢的图标：
     - SVG Repo 搜索“flag icon”：https://www.svgrepo.com
     - Wikimedia Minesweeper_flag.svg（经典扫雷旗帜）：https://commons.wikimedia.org/wiki/File:Minesweeper_flag.svg

2. 地雷图标：命名为 mine.svg
   - 可在这些网站搜索“bomb”或“mine icon”：
     - SVG Repo：https://www.svgrepo.com
     - Flaticon：https://www.flaticon.com

将下载好的 SVG 文件重命名为：
   - flag.svg
   - mine.svg

并放到当前 images 目录下，刷新网页即可在游戏中看到旗帜和地雷图标。
